关于我

发表于 2016-10-14

我叫王大锤相信用不了多久我就会升职加薪当上总经理出任CEO迎娶白富美走向人生巅峰想想还有点小激动我记得这个还有一个奔跑和开头的两个，我们要去远方看看还有什么是我们的思密达；我的生涯一片无悔，我想起那天夕阳下的奔跑，那是我逝去的青春。想要营救公主，就要，嗯..额..这...就要...就要...就要从我的尸体上跨过去...拜托能把台词背熟再上来吗...难道是怕我的销魂的姿势闪瞎他的狗眼。我叫王大锤，万万没想到将军竟然主动去世了不过这并不会影响我一代名医的招牌······吧。我感受到一股强大的力量正在向双拳汇聚。我的左拳像火焰一样炙热，我的右拳像冰霜一样寒冷。左右拳字幕出现，左拳字幕：火焰特效;右拳字幕：冰霜特效)王大锤：我了个大叉，什么玩意啊?连特效都没做，直接打字也太敷衍了吧?敌人甲：竟然是九阳神拳之——冰火九重天?!王大锤：你怎么看出这招来的?这名字太色情了吧?

以上纯属瞎掰，关于我，是一个生于济南的90后纯汉子。从小就特立独行，喜欢一个人瞎琢磨。初中那会就梦想着做一名网络安全工程师，那个时候大概是2005年把，你一年我15岁。

初次接触计算机是在学校旁边的小网吧，跟着高年级的一起去玩，那会特紧张，满网吧的CS，当时觉得特牛逼，心想要是自己也能写出一个这样的游戏该多好，是的我跟别人想的不一样，不是想着怎么玩，而是想着怎么做。

学校那会有开设计算机基础课，进机房还一人带个鞋套，也就是教教打字啊一类，依旧记得当时电脑都带着office，我就发现里面有一个特奇怪的名字叫做“宏”，自己搞不懂就跑去问计算机老师，老师当然是没理我。那时候傻不晓得google一下......如同有次去问物理老师“力是怎么产生的？”，老师答曰这个问题也没搞明白......是的我从小就奇葩。

18岁后的日子

高中刻苦读了三年，上了一所普通高校，读了个地质工程。是的你没有听错，是地质工程而不是计算机。本着干一行爱一行的思想大学毕业后在地质行业做了两年。在大家觉得我会在这行做一辈子的认知中我辞职了。就是这么任性。我决定要重新投入互联网额怀抱。技术在飞速的发展，各种新的系统、语言、框架、技术层出不穷。于是乎由浅入深开始收复失地。发现原来的网页三剑客已成过去。网站建设也分离出了UI、前端、后端，于是便从前端开始着手。HTML CSS ES 重新学起，发现单纯这些远远不够，从用户打开浏览器到页面在浏览器发生了什么？不同浏览器内核有何差异？万恶的IE6兼容性如何解决？HTML5 有哪些新特性，ES6多了那些新标签？响应式框架如何搭建？原生css实现后有没有造好的车轮子（bootstrap fundation semanticUI pure）？除了手写JS也有现成优秀的框架来学习（Jquery angular vue.js）。 SASS编译、Nood.js、apache 、PHP、 mysql不断学习新的知识。写的有点乱后续再写----LOL烂尾王的自白
